# 教程：安装并初始化 IPFS
这些课程已经在 go-ipfs 版本0.5.0上进行了测试。_请在github上更新此文件，以反映已经测试过的其他版本。_

## 先决条件

- 你应该对命令行有一定的了解

## 学习目标
这些课程将教你如何

* 安装IPFS
* 初始化一个IPFS仓库
* 定位IPFS存储你的本地IPFS仓库内容的位置
